# Backend - Python API Server

This is the Python backend for UsenetSync.

## Structure
- `src/unified/` - Main unified system code
- `src/legacy/` - Old code (to be removed)
- `tests/` - Backend tests

## Running
```bash
cd /workspace
source venv/bin/activate
python start_backend.py
```

## Testing
```bash
cd backend/tests
pytest
```
